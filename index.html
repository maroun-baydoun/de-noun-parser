de-noun-parser

Parse Wikitext's de-noun template

Github NPM

1. Install

npm i de-noun-parser
Or
yarn add de-noun-parser

2. Use

import parse from "de-noun-parser";

const wikitext = `
===Noun===
{{de-noun|m||Honige}}
# [[honey]]
`;

const parsed = parse(wikitext);
